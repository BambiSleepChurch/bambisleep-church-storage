‚ö° Quick Start

1

Clone & Install

Clone the repository and install dependencies

git clone https://github.com/BambiSleepChurch/bambisleep-church-storage.git cd bambisleep-church-storage npm install
2

Build the Server

Compile TypeScript to JavaScript

npm run build
3

Configure Your MCP Client

Add to your Claude Desktop or VS Code MCP configuration

{ "mcpServers": { "brandyfication": { "command": "node", "args": ["/path/to/dist/index.js"], "env": { "STORAGE_DIR": "/path/to/BRANDYFICATION" } } } }
4

Start Using!

The BRANDYFICATION tools are now available in your MCP client. Try: "Upload an image" or "List all images"
